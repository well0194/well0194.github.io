Information about some of the tags

Please note, I did not add the exact terms and citations as it would change the website. Since this just an example of how each tag would be used.

  1. img scr - This tag is for adding an image to a website
  2. table - This tag is for adding a table
  3. strong - This tag is for a more bolded text
  4. abbr title - This is for a link attached to a certain word
  5. mark - This is for a highlighted term
